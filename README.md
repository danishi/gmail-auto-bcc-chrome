# Gmail Auto BCC

Gmailでメールを作成する際、アカウントごとに設定したメールアドレスを自動的にBCCに追加するChrome拡張機能です。

## インストール方法

1. このリポジトリをクローンまたはダウンロードします
   ```bash
   git clone https://github.com/danishi/gmail-auto-bcc-chrome.git
   ```
2. Chromeで `chrome://extensions` を開きます
3. 右上の「デベロッパーモード」を有効にします
4. 「パッケージ化されていない拡張機能を読み込む」をクリックします
5. クローンしたディレクトリを選択します

## 使い方

1. インストール後、自動的に設定ページが開きます（または拡張機能の「オプション」から開けます）
2. 「アカウントを追加」をクリックします
3. Gmailアドレスと、自動追加したいBCCアドレスを入力します
4. 「保存」をクリックします

以降、Gmailで新規メール作成・返信・転送時にBCCが自動的にセットされます。

## 機能

- Gmailの新規作成、返信、転送すべてに対応
- アカウントごとに異なるBCCアドレスを設定可能
- 1つのアカウントに複数のBCCアドレスを設定可能
- 有効/無効の切り替え
- 同じアドレスが既にBCCに設定されている場合は重複追加しない
- BCC入力後、自動的に適切なフィールド（本文・宛先など）にフォーカスを移動

## 技術仕様

- Chrome Manifest V3
- アカウント自動検出によるマルチアカウント対応
- `chrome.storage.sync` による設定の同期保存
